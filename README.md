# 🚀 Fullstack TurboRepo App

A modern scalable fullstack monorepo application built using:

- 📱 React Native
- ⚡ Tamagui UI
- 🌐 Next.js Dashboard
- 🔥 Firebase
- 🐍 FastAPI Backend
- 🧠 Redux Toolkit
- 🏗️ TurboRepo

# 🚀 Getting Started

## Installation

```bash
pnpm install
```

## Run Mobile App

```bash
pnpm mobile
```

## Run Dashboard

```bash
pnpm dashboard
```

## Run Backend

```bash
pnpm api
```
